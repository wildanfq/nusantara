# Wildan Firmani Quraisi
**IT Infrastructure Engineer** | 20 Tahun

---

Sebagai seorang IT Infrastructure Engineer, saya berdedikasi untuk membangun dan memelihara ekosistem teknologi yang aman, tangguh, dan berperforma tinggi. Fokus utama saya terletak pada otomatisasi, keandalan sistem, dan arsitektur jaringan yang solid.

### Keahlian Utama (Core Competencies)

* **Modern Server Operations:** Spesialis dalam administrasi Linux dan orkestrasi beban kerja menggunakan Docker. Saya menerapkan prinsip *Infrastructure as Code* (IaC) untuk memastikan konfigurasi server yang konsisten, efisien, dan dapat direplikasi dengan cepat.
* **Advanced Networking & Security:** Mahir merancang jaringan menggunakan Cisco dan Mikrotik, termasuk implementasi VPN, VLAN, dan *firewall hardening*. Saya terbiasa menyimulasikan arsitektur sebelum tahap produksi guna meminimalkan risiko *downtime*.
* **Full-Stack Observability:** Menguasai sistem pemantauan berbasis Prometheus dan Grafana untuk mendeteksi anomali secara *real-time*, mengoptimalkan penggunaan sumber daya, dan menjaga ketersediaan layanan (*uptime*).
* **Hardware Lifecycle Management:** Ahli dalam *troubleshooting* perangkat keras tingkat lanjut, perbaikan komponen, serta perencanaan kapasitas infrastruktur fisik untuk mendukung kebutuhan operasional jangka panjang.

### Eksplorasi Sistem Fundamental (Low-Level Systems)

Di luar peran operasional infrastruktur, saya memiliki ketertarikan mendalam untuk membedah bagaimana komputer bekerja di level paling esensial:

* **Hardware Design:** Mendesain PCB menggunakan KiCad dan merancang simulasi sirkuit logika digital secara mendalam—mulai dari *register* hingga komponen CPU dasar—melalui Logisim Evolution.
* **Processor Architecture:** Mendalami arsitektur RISC-V melalui implementasi menggunakan Verilog (HDL) dan pemanfaatan *open-source toolchain* seperti OpenLane.
* **Systems Programming:** Mengembangkan sistem operasi (OS) minimalis menggunakan bahasa C dan Zig. Fokus eksplorasi saya adalah pada manajemen memori tingkat rendah dan interaksi presisi antara perangkat lunak dan perangkat keras.
